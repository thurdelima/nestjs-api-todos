## Run Locally

Clone the project

```bash
  git clone https://github.com/thurdelima/nestjs-api-todos
```

Go to the project directory

```bash
  cd nestjs-api-todos
```

Install dependencies

```bash
  npm install 
```

Configure .ENV vars

And

```bash
  npm run start:dev

```
